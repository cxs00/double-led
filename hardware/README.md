## 🎯 硬件约束

### 🔌 STM32F103C8T6引脚分配
- **调试接口**: PA13 (SWDIO), PA14 (SWCLK)
- **SPI2接口**: PB13 (SCK), PB14 (MISO), PB15 (MOSI), PB7 (LATCH)
- **电源**: 3.3V供电，最大工作电流150mA

### 🔗 74HC595级联连接
- **U1**: D7段码控制 + LED阴极控制
- **U2**: 位选择 + LED阳极控制 (PNP晶体管反相)
- **U6**: D8段码控制
- **级联顺序**: STM32 → U1 → U2 → U6

### 📡 74HC165按键扫描
- **9位按键扫描**: SER + D0-D7
- **按键映射**: RUN/STOP, UP/DOWN, PRG, ENTER, SHIFT, FUNC, 保留2个
- **扫描频率**: 100Hz

### ⚡ SPI2接口定义
- **时钟频率**: 9MHz
- **数据位宽**: 8位
- **传输模式**: 全双工
- **时序**: 标准SPI时序

### 🐛 SWD调试接口
- **接口类型**: Serial Wire Debug
- **连接器**: 2.54mm间距排针
- **调试器**: J-Link, ST-Link兼容
