#### 🔧 技术规格
```
技术规格:
├── 主控芯片: STM32F103C8T6 (ARM Cortex-M3, 72MHz)
├── 显示驱动: 3×74HC595级联 (24位输出)
├── 按键扫描: 1×74HC165 (9位输入)
├── 通信接口: SPI2 (9MHz全双工)
├── 调试接口: SWD (Serial Wire Debug)
├── 电源系统: 3.3V单电源供电
└── 工作温度: -40°C ~ +85°C
```

#### 📐 硬件规格
```
硬件规格:
├── 显示系统: 双5位数码管显示
├── LED指示: 6个状态指示LED
├── 按键系统: 9键按键矩阵
├── 调试接口: SWD调试接口
├── 电源要求: 3.3V ±5%, 最大150mA
└── 尺寸规格: 标准PCB尺寸
```

#### 🔌 引脚分配
```
引脚分配:
├── 调试接口: PA13 (SWDIO), PA14 (SWCLK)
├── SPI2接口: PB13 (SCK), PB14 (MISO), PB15 (MOSI), PB7 (LATCH)
├── 电源引脚: VCC (3.3V), GND (地)
├── 晶振引脚: PA8 (MCO), PA9 (OSC_IN)
└── 使用率: 21.6% (8/37引脚)
```

#### ⚡ 信号规格
```
信号规格:
├── SPI2信号: 9MHz时钟, 3.3V CMOS
├── 显示信号: 动态扫描, 1000Hz刷新
├── 按键信号: 100Hz扫描, 10ms去抖动
├── 调试信号: 最高10MHz, SWD协议
```
