# 快速开始指南

## 🚀 5分钟快速上手

### 第一步：硬件连接
```
STM32F103C8T6 ←→ 74HC595×3 ←→ 数码管D7/D8
     ↓
   74HC165 ←→ 按键矩阵
     ↓
   J-Link ←→ 调试接口
```

### 第二步：软件准备
1. **打开Keil项目**
   ```
   路径：software/MDK-ARM/500 double led.uvprojx
   ```

2. **编译项目**
   ```
   按F7或点击Build按钮
   等待编译完成（0 Error, 0 Warning）
   ```

3. **配置调试器**
   ```
   Options for Target → Debug → J-Link / J-Trace Cortex
   Settings → Debug → Port: SW (不选JTAG)
   ```

### 第三步：程序烧录
1. **连接J-Link**
   - SWDIO → PA13
   - SWCLK → PA14
   - GND → GND
   - VCC → 3.3V

2. **烧录程序**
   ```
   点击Load按钮
   等待烧录完成
   ```

### 第四步：功能测试
1. **上电测试**
   - 数码管显示：D7显示频率，D8显示电流
   - LED指示：Hz LED点亮
   - 默认频率：50.00Hz

2. **按键测试**
   - UP键：频率递增
   - DOWN键：频率递减
   - FUNC键：切换高低频模式
   - RUN键：运行指示

### 常见问题快速解决
1. **编译错误** → 检查头文件包含
2. **烧录失败** → 检查SWD连接和Flash算法
3. **显示异常** → 检查数码管连接和共阳极逻辑
4. **按键无响应** → 检查74HC165连接
