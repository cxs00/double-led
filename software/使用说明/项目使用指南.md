# STM32 500双LED项目使用指南

## 🎯 项目概述

STM32 500双LED项目是一个基于STM32F103C8T6的变频器控制系统，具有10位数码管显示和6个状态LED指示功能。

## 🚀 快速开始

### 1. 硬件准备
- STM32F103C8T6开发板
- 74HC595×3（显示驱动）
- 74HC165（按键输入）
- 10位数码管（D7+D8）
- 6个状态LED
- J-Link调试器

### 2. 软件准备
- Keil uVision 5.25+
- STM32CubeMX
- J-Link驱动

### 3. 项目编译
1. 打开 `software/MDK-ARM/500 double led.uvprojx`
2. 编译项目（F7）
3. 检查编译结果

### 4. 程序烧录
1. 连接J-Link调试器
2. 配置调试接口为SWD
3. 烧录程序到STM32

## 🎮 功能使用

### 显示功能
- **D7显示**：频率信息（XXX.XX Hz 或 XXXX.X Hz）
- **D8显示**：电流信息（XXX.XX A）
- **LED指示**：ALM、V、F/R、Hz、RUN、A

### 按键功能
- **UP/DOWN**：频率调节，支持长按连续调节
- **FUNC**：高低频模式切换
- **RUN/STOP**：运行控制
- **其他键**：菜单导航和参数设置

### 频率控制
- **低频模式**：0.00-400.00Hz，步进0.01Hz
- **高频模式**：0.0-4000.0Hz，步进0.1Hz
- **智能步进**：4级步进系统

## 🔧 操作说明

### 基本操作
1. **上电启动**：系统自动初始化
2. **频率调节**：使用UP/DOWN键调节
3. **模式切换**：使用FUNC键切换高低频
4. **运行控制**：使用RUN/STOP键控制

### 高级功能
1. **智能步进**：自动步进级别升级
2. **长按调节**：长按UP/DOWN连续调节
3. **菜单导航**：使用PRG/ENTER/SHIFT导航
4. **参数保存**：设置参数自动保存

## ⚠️ 注意事项

### 硬件注意事项
- 确保所有连接正确
- 检查电源电压（3.3V/5V）
- 注意共阳极数码管特性

### 软件注意事项
- 使用SWD调试接口
- 避免使用JTAG相关GPIO
- 注意Flash算法配置

## 🐛 故障排除

### 常见问题
1. **显示异常**：检查数码管连接
2. **按键无响应**：检查按键连接
3. **频率异常**：检查参数设置
4. **系统死机**：重新上电复位

### 调试方法
1. 使用Keil调试器查看变量值
2. 用示波器检查SPI波形
3. 单独测试显示和按键功能
4. 逐步调试智能步进逻辑
