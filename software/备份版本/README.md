# 备份版本

## 📋 文件夹说明

本文件夹包含STM32 500双LED项目的备份版本文件。

## 📚 备份内容

### 历史版本
- **v1.0** - 基础版本（2025-10-25）
- **开发版本** - 开发过程中的重要版本
- **测试版本** - 功能测试版本

### 备份类型
- **完整备份** - 包含所有源代码和文档
- **增量备份** - 仅包含变更的文件
- **配置备份** - 项目配置文件备份

## 🔄 版本管理

### 版本命名规则
```
格式：500_double_led_vX.Y.Z
示例：500_double_led_v1.0.0
```

### 版本类型
- **主版本号（X）**：重大功能更新
- **次版本号（Y）**：功能改进和优化
- **修订号（Z）**：问题修复和小改进

## 📦 备份策略

### 自动备份
- 每次重大修改后自动创建备份
- 保留最近10个版本
- 自动清理过期备份

### 手动备份
- 重要修改前手动备份
- 重要功能测试前创建测试备份
- 发布前创建发布备份

## 🎯 使用说明

### 恢复版本
1. 选择要恢复的版本
2. 解压备份文件
3. 替换当前文件
4. 重新编译项目

### 版本对比
1. 使用版本对比工具
2. 查看变更记录
3. 分析影响范围
4. 决定是否恢复

## 📁 备份文件结构

```
备份版本/
├── v1.0.0/                    ← 版本1.0.0备份
│   ├── Core/                  ← 源代码
│   ├── Drivers/               ← HAL库
│   ├── MDK-ARM/               ← Keil项目
│   └── 版本说明.txt           ← 版本说明
├── 开发版本/                  ← 开发版本备份
└── 测试版本/                  ← 测试版本备份
```
