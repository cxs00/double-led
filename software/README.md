## 🚀 快速开始

### 1. 打开项目
```
路径：config/MDK-ARM/500 double led.uvprojx
工具：Keil uVision 5
```

### 2. 编译项目
```
快捷键：F7
或点击：Build按钮
检查：0 Error, 0 Warning
```

### 3. 配置调试器
```
Options for Target → Debug → J-Link / J-Trace Cortex
Settings → Debug → Port: SW (不选JTAG)
Flash Download → Programming Algorithm: STM32F10x Med-density (64KB)
```

### 4. 烧录程序
```
连接J-Link调试器
点击Load按钮
等待烧录完成
```

## ⚠️ 注意事项

### 硬件注意事项
- 确保所有连接正确
- 检查电源电压（3.3V/5V）
- 注意共阳极数码管特性

### 软件注意事项
- 使用SWD调试接口
- 避免使用JTAG相关GPIO
- 注意Flash算法配置

## 🐛 故障排除

### 常见问题
1. **显示异常** - 检查数码管连接
2. **按键无响应** - 检查按键连接
3. **频率异常** - 检查参数设置
4. **系统死机** - 重新上电复位

### 调试方法
1. 使用Keil调试器查看变量值
2. 用示波器检查SPI波形
3. 单独测试显示和按键功能
4. 逐步调试智能步进逻辑
