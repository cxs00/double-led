## 🚀 快速配置

### 1. 开发环境
- 安装Keil uVision 5.25+
- 安装STM32CubeMX
- 安装J-Link驱动

### 2. 硬件连接
- 连接STM32F103C8T6
- 连接74HC595×3
- 连接74HC165
- 连接J-Link调试器

### 3. 软件配置
- 打开Keil项目
- 配置调试器
- 设置Flash算法
- 编译和烧录

## ⚠️ 重要配置

### 调试器配置
```
调试器：J-Link / J-Trace Cortex
接口：SWD (不选JTAG)
端口：SW
```

### Flash算法配置
```
算法：STM32F10x Med-density (64KB)
地址：0x08000000
大小：0x10000
```

### SPI配置
```
模式：Master
数据位：8位
时钟极性：Low
时钟相位：1Edge
波特率：8MHz
```
